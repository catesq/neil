Using Pattern Thrus (Effects Only)

To allow generators to pass through an effect unchanged by that effect, you may use a Pattern Thru. This will temporarily disable the effect while letting the generators play through.

To insert a Pattern Thru, navigate to the desired row and column, then press the _ (underscore) key . The word <Thru> will now appear in the row (as shown in Figure 3.7).

A Pattern Thru is disabled when the sequence runs into a <break>.


It it worth mentioning that not all effects work with the thru command. While it was in the original documentation regarding coding machines, it was missed on various occasions. Jeskola, newer, and future machines should make better use of the Pattern Thrus.
