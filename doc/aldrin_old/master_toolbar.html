Master Toolbar
The master toolbar lets you alter various global song elements.
 
#
Name Description
1
Master Gain This small slider lets you adjust the volume for the entire song. The black bars above and below this slider are the VU meters for the song indicating real-time amplitude changes.
2
BPM This box allows you to change the current Beats Per Minute of this song.
3
TPB This box allows you to change the current Ticks Per Beat of this song.
