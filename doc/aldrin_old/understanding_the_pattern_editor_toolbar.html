Elements of the Pattern Editor Toolbar
Once you are in the Pattern Editor ( Press F2 ) you will be presented with a new toolbar, just below the global toolbar. The following represents the Pattern Editor toolbar. (Note: The checkered bars indicate a cut and re-connection of the toolbar).
Figure 2.1
Toolbar Elements:

1. Current Machine : This drop-down box indicates the currently selected Machine . Use this to change the current machine. This may also be accomplished with the Ctrl+ and Ctrl- keys.

2. Current Pattern : Each machine will most likely have multiple patterns associated with it. The value of this select box indicates the current pattern. This may also be changed by pressing the + (plus) and - (minus) keys of the keypad on your keyboard.

3. Current Wave File : This contains the list of loaded wave files (done in the wavetable editor), and indicates the currently selected one. When working in a Tracker machine, changing this will automatically change the sample as you add notes.

4. Base Octave: The base octave indicates the current range of keys (octaves) you have your keyboard set to enter. This number may be changed from 0 to 9, with 0 being extremly low notes, and 9 being extremly high notes.

5. Play Notes : The play notes checkbox is useful when you want to hear the notes as you enter them with your keyboard.
