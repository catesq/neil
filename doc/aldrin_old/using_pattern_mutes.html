Using Pattern Mutes
Much like the Pattern Break , Pattern Mutes stop the current pattern, yet unlike the Pattern Break which allows the final note to play through, the Pattern Mute completely mutes all audio in that channel until another pattern is called. A new pattern will function as a break. So sometimes you will notice that the last notes before the mute will continue playing after the new pattern is called. This will not happen when there are new notes replacing the ones from before the mute.

To insert a Pattern Mute, navigate to the desired row and column, then press the - (minus) key . The word <mute> will now appear in the row (as shown in Figure 3.6).

Note Muting an effect will effectively mute all generators going into it. If you would rather disable the effect, yet let the generators continue through unchanged - use a Pattern Thru instead.
