Understanding the Wavetable Editor
The following screen represents the Wavetable Editor in Aldrin. The red markers indicate its 5 distinct functional areas.
Wavetable Editor Elements:
#
Element Description
1
Song Wavetable This area contains a list of samples which you have loaded into this song.
2
Tool Buttons Various tool buttons allow you to perform various operations specific to the Wavetable Editor.
3
Sample Library This area shows the available samples on your local drive. It shows all WAV and XI files.
4
Instrument Information This area gives you various information about the currently selected instrument
5
Sample Envelope This area lets you define, load, and save complex envelopes to use on samples
