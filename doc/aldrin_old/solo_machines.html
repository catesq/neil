Solo Machines

To solo a machine so that it is the only machine currently emitting sound, either double click on the LED in the upper left-hand corner, or right-click and select solo from the menu. Square brackets [ ] appear around all other machines, indicating that the machine you selected is currently in solo mode. A check will also appear next to the word Solo in the right-click pop-up indicating this status.
