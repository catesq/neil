Signal Analysis

Figure 3.12

Signal Analysis in Aldrin presents a simple way to obtain a visual display of the audible interaction between a source and destination machine.

To obtain the Signal Analysis display for a specific connection, simply right-click on the arrow between them, and select Signal Analysis .


The windows:

The time window shows a real-time rhythmic display allowing the user to judge both audio source rhythm and amplitude.

The frequency window shows a spectral representation of the active frequencies emitting from the source machine.
