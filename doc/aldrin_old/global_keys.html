Global Interface
The following keys are available anywhere throughout Buzz.
Name Key Description
New CTRL-N Creates a new Aldrin song
Open CTRL-O Opens a Aldrin file ( BMX or BMW )
Save CTRL-S Saves the current file.
Exit ALT-F4 Exits Aldrin.
Cut CTRL-X Cuts current values in the pattern editor, and patterns in the Sequence Editor.
Copy CTRL-C Copies current values in the pattern editor, and patterns in the Sequence Editor.
Paste CTRL-V Pastes current values in the pattern editor, and patterns in the Sequence Editor.
  Help F1 Loads the Aldrin User Manual
Pattern Editor F2 Used for adding patterns, adding notes/parameter changes to notes.
Machine Editor F3 Used to add and route machines.
Sequence Editor F4 Used to devise pattern order, forming sequences.
Play F5 Plays Song or loop from beginning or from the point you stopped it.
  Play from Poisition F6 Plays the song from the current cursor position
Record F7 Records Parameter changes to patterns.
Stop F8 Stops song or loop.
Loop N/A Enables song looping.
Wavetable F9 Allows you to add and manage samples and instruments in your songs.
User Notes F10 Allows you to add notes about this song.
  Toggle Dialogs F11  
Free Wave F12 Frees the Audio device for other apps, instantly muting all audio in Aldrin (Panic!)
