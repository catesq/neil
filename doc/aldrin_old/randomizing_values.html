Randomizing Values
Adding random values to columns and rows is an easy way to produce a huge variety of sequenced sounds. This is done by simply selecting an area , and pressing Ctrl-R . Nice toying is setting random values for all the parameters in a pattern of for example the ZX7, which has a very neat Graphic User Interface. Then play the pattern, open the GUI by selecting Edit after right-clicking the machine in the machine editor, and watch all the knobs and wheels twist and turn like maniacs.
