Editing Envelope Points
Designing envelopes is a very simple task in Buzz. Envelopes are based on a serious of lines separated by points. You create an envelope in Aldrin by adding points and dragging their handles into place. The following tasks enable you to do this:

Add an Envelope Point

To add an envelope point left-click anywhere within the black area of the Envelope Editor. A new handle will appear in the spot you clicked.

For future versions:

Edit Points

To edit an envelope point left-click its handle and drag it into a new position. If you would like more precise editing, right-click on it and select Edit Point from the pop-up. This will allow you to type X and Y coordinates for the point to be placed.


Delete Point

To delete a point left-click on its handle, and select Delete point from the pop-up
