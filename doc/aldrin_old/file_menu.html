File Menu

Aldrin has the very familiar File menu if you've worked previously with just about any program. The following items are available:

Command Key Description
New CTRL-N Creates a new file. Prompts to save existing work.
Open CTRL-O Open a new file. Promts to save existing work.
Save CTRL-S Save current song.
Save As N/A Save with a new name.
Recent Files N/A A list of recent files you've worked with which may be opened by selecting them.
Exit ALT-F4 Quit Aldrin
