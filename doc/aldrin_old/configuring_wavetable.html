Configuring Wavetable
In order to use samples within Aldrin you must configure Wavetable directories. Once a directory is added, that directory and all of its sub-directories may be accessed from Aldrin Wavetable , and thus all sample based machines.
Adding Wavetable Directories

To add a Wavetable directory simply click the Add button shown above, and select a folder from your system from the pop-up window. You may add and delete Wavetable directories at any time.
