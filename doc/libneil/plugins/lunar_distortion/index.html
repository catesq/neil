Lunar Distortion

Function Choose a waveshaping function that will alter the audio signal by computing function output for each of the input samples. You can choose between the following functions.
  • HyTan
  • Assym
  • Sinus
Gain In (dB) Gain specified in decibel by which the signal is amplified before being passed to the waveshaping function.
Min: 0.0dB
Max: 24.0db
Gain Out (dB) Gain specified in decibel by which the signal is amplified after being passed to the waveshaping function.
Min: -24.0dB
Max: 0.0db
