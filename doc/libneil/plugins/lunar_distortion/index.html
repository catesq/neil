Lunar Distortion

Function Choose a waveshaping function that will alter the audio signal by computing function output for each of the input samples. You can choose between the functions described in the end of this help document. Currently the functions are as listed.
  • HyTan
  • Assym
  • Sinus
Gain In (dB) Gain specified in decibel by which the signal is amplified before being passed to the waveshaping function.
Min: 0.0dB
Max: 24.0db
Gain Out (dB) Gain specified in decibel by which the signal is amplified after being passed to the waveshaping function.
Min: -24.0dB
Max: 0.0db

Waveshaping Functions

HyTan Hyperbolic tangent function
Assym Assymetric function
Sinus Sine function
