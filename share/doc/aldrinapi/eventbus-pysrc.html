  4  Supplies all user interface modules with a common bus 
  5  to handle status updates. 
  6   
  7  this is the order of operation: 
  8   
  9  1. on first time loading, GlobalEventBus registers all possible event types (add new ones below). 
 10  2. main window instantiates all other ui objects 
 11  3. no other ui object may register any new event id. 
 12  4. upon creation, all ui objects listening to events must connect their handlers using  
 13     eventbus.<event name> += <function>[,<arg1>[,...]] 
 14  5. call events using 
 15     eventbus.<event name>([<arg1>[,...]]) 
 16   
 17  please note: 
 18
