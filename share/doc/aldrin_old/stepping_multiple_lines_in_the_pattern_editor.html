Stepping Multiple lines in the Pattern Editor

A nice feature in the Aldrin Pattern Editor is the ability to skip multiple pattern lines each time a new note is entered. For instance, if you wanted to place a Kick drum every 4 steps, you would set the line step to 4.

Setting a line step is done by pressing CTRL-[Number of Lines to Skip] . In other words, if you wish the cursor to junp 8 lines further, each time a value is entered, press CTRL-8 . To return to the default of 1 step per value, press CTRL-1 .

There is no drop-down menu or anything like that for this feature in Aldrin. I am afraid only the real die-hards in reading manuals will reach this topic and will know about this feature. Well.. spread the news !
