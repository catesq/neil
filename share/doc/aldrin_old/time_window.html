Time Window
The Time window is a very simple way to see the various times related to your song.
The following elements exist on the Time window (left to right):

#
Name Description
1
Elapsed This indicates the elapsed time since you pressed play
2
Current This holds the same value as the 'elapsed' time, but will be reset as soon as you reach the end of a loop, where the elapsed time keeps on counting.
3
Loop This indicates the length of the loop.
