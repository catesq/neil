Sequence Editor (F4)
The following keys are available in the Sequence Editor
Key Command Description
Arrow Keys Move Cursor Move selection cursor one space.
SHIFT + Arrow Keys Select Area Selects rows and columns.
SHIFT + Left Click Select Area Selects rows and columns.
PAGE UP/DOWN Move Page Moves the cursor 16 bars up or down.

. (period)

Clear

Clears field under cursor

Delete

Delete

Deletes row under cursor

Enter

Edit pattern

Switches to current pattern in the Pattern Editor

Insert

Insert

Inserts an empty row (in the current column).

CTRL+I

Insert Row

Inserts a row in the entire sequence.

CTRL+D

Delete Row

Deletes a row from the entire sequence.

CTRL+X

Cut

Cuts the current selection to the clipboard.

CTRL+C

Copy

Copies the current selection to the clipboard.

CTRL+V

Paste

Pastes at cursor (overwriting)

CTRL+RETURN

New Track

Creates new track/column

CTRL+DELETE

Remove Track

Removes the current track.

CTRL+L

Solo

Solos the current machine (generators only) .

CTRL+Right Arrow

Move Track Right

Moves the current column to the right.

CTRL+Left Arrow

Move Track Left

Moves the current column to the left.
