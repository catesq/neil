About the Pattern Editor
The Aldrin Pattern Editor is used to create the instructions for the machines that create sound in Aldrin. As its name indicates, the Pattern Editor serves the single purpose of creating and editing items called ' patterns' which are used to control the output of machines.

Once patterns are created, they are then sequenced to form longer portions of music - and eventually full songs. Consider patterns to be the script of seperate scenes in a movie, or the subroutines of a program.

To access the Pattern Editor, Press F2 or click the Pattern Editor icon on the global toolbar.
