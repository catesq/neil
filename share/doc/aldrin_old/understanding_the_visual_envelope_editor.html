Understanding the Visual Envelope Editor

The following indicates the available features and elements of the Wavetable Envelope Editor:

#
Element Description
1
Sample File Location The physical hard disk location of the currently selected sample.
2
Sample Gain This slider allows you to adjust the master volume of this sample.
3
Loop Options To enable a sample looping .
4
Machine Name This drop-down box allows you to select a machine to apply an envelope to. It will include all machines within the current song that support Wavetable Envelopes.
5
Envelope Name This is the specific envelope name for this machine. Machines may have many types of envelopes including Pan, Pitch, and Filter envelopes.
6
Envelope Disable Select this to disable this envelope.
7
Graphical Envelope Editor This allows you to edit the various points for this envelope.
