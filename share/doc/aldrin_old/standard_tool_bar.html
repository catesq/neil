Standard Tool Bar
The standard toolbar allows easy access to many of the most important screens and functions of Aldrin. This toolbar may operate as both a standard or docked toolbar, sticking itself to any side of your screen.
The following elements appear in the standard menu bar (left to right):
Name Key Description
File
New CTRL-N Creates a new Aldrin song
Open CTRL-O Opens a Aldrin file ( BMX or BMW )
Save CTRL-S Saves the current file.
Clipboard
Cut CTRL-X Cuts selected values in the pattern editor, and selected patterns in the Sequence Editor.
Copy CTRL-C Copies selected values in the pattern editor, and selected patterns in the Sequence Editor.
Paste CTRL-V Pastes selected values in the pattern editor, and selected patterns in the Sequence Editor.
Primary Navigation
Pattern Editor F2 Used to add and edit patterns.
Machine Editor F3 Used to add and rout machines.
Sequence Editor F4 Used to devise pattern order, forming sequences.
Song
Play F5 Plays Song or loop from the beginning or the point you pressed stop.
Record F7 Records Parameter changes to patterns.
Stop F8 Stops (pauses) song or loop.
Loop N/A Enables song looping.
Secondary Navigation  
Wavetable F9 Allows you to add and manage samples and instruments in your songs.
User Notes F10 Allows you to add notes about the current song.
Utility  
Free Wave F12 Frees the Audio device for other apps instantly muting all audio in Aldrin (Panic!)
