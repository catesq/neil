Naming and Renaming Machines
Machine names can be very useful toward organizing songs in Aldrin. Due to the fact that so many machines exist, which each have a wide spectrum of sound creation possibilities, it can be extremely useful to name a machine according to what it is being used for. A complex synth (ex. M4) emulating a 303 could be named "303" to make it easy to remember. This changes the machines identification on not only the Machine Editor , but the Pattern Editor and Sequencer Editor as well. Renaming machines can be done at any time by right-clicking, and selecting Rename ... . from the pop-up menu.
