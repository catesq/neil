Connecting Machines
Connecting machines for output is simple.
 
1. Hold down shift, left-click on a machine and do not release.
2. While keeping the button down, move cursor to second machine
3. Release the button.
The machines should now be connected.
Please note the following rules and exceptions:
1. Nothing can connect into a generator - the function of a generator is to create sound. Generators begin every chain.
2. For connecting to mono-effects, a stereo machine output must either be split into two separate mono machine inputs, or the master out itself. Stereo inputs are not currently supported for every effect.
