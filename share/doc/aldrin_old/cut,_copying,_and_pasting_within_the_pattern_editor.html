Cut, Copying, and Pasting within the Pattern Editor
Working with the clipboard in the Pattern Editor is much like doing the same in any other application:

First, make a selection

To copy the selection:

Either press CTRL-C or select "Copy" from the Right-Click pop-up menu .

To cut the selection:

Either press CTRL-X or select "Cut" from the Right-Click pop-up menu .

To Paste the selection:

Either press CTRL-V or select "Paste" from the Right-Click pop-up menu



Note that pasting values from certain columns to columns that do not represent the same parameters is not possible.

Copying and pasting notes from the note columns of one machine to the note columns of another machine on the contrary, -is- possible.

MIDI-users, notice that this makes it possible to record notes from your MIDI-keyboard into the patterns of a MIDI-supporting machine and eventually pasting the notes into a pattern of another machine.
