The Aldrin Interface

To effectively use Aldrin you must first learn to manoeuvre throughout its interface and quickly use its global commands. This includes understanding not only Aldrin's various sections, but also understanding how important keyboard short-cuts are to be able to work efficiently.

The global interface of Aldrin is composed of the following areas:

Menus

  1. File
  2. Edit
  3. View
  4. Help

Toolbars

  1. Standard
  2. Master
  3. Time
