View Menu
The view menu allows you to access various utilities and configuration options within Aldrin. Note the following:

Command Description
CPU Monitor Opens a window containing CPU usage information broken down by machines.
Hard Disk Recorder Allows you to record your work directly to disk.
Master Toggles the Master floating (or docked) tool bar.
Status Bar Enables or disabled the lower status bar.
Time Toggles the Time floating (or docked) window.
Standard Toggles the Standard floating (or docked) tool bar.
Themes Allows you to switch between various color themes for your Aldrin GUI.
Preferences All Aldrin User Preferences are stored here.
