Editing Tracks in Pattern Editor
Adding Tracks

Adding tracks is accomplished by pressing Ctrl + , or Right-Clicking and Selecting "Add Track" from the pop-up menu. As mentioned earlier, added tracks differ from machine to machine. Some machines (such as the Voidsynth) only duplicate the velocity and note columns, while others (such as the Jeskola Delay) duplicate all of their parameters.

Deleting Tracks

If you wish to delete a track, press Ctrl - (minus) or Right-Click and Select "Delete Last Track" from the Pop-up. Note that you must delete the last-most previously created track. This means if you have three tracks, and you wish to delete the second track, you must delete the third first. Also note that this function is permanent. There is no Undo.
