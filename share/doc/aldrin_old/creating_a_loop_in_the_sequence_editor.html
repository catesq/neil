Creating a Loop in the Sequence Editor

Loops are useful regions set within a song, in order to get smaller areas to loop through while editing one portion of a song.

Setting the Loop Begin Point

To set the Begin Point for a loop, move your cursor to the desired row and press CTRL-B . This will place a thin horizontal hashed black line indicating that the song will begin there when played. Yes, setting a loop begin point is the same as setting a song begin point in Aldrin.

Setting the Loop End Point

To set a Loop End Point, move your cursor to the desired row and press CTRL-E One Time . (Pressing it two times sets a Song end point). This will place a hashed horizontal line indicating that the loop end point has been placed.
